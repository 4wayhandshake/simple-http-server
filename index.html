Simple HTTP Server

This is a page for all your simple serving needs.

Perfect for nothing really, but easy to deploy! 
usage: SimpleServer.py [-h] [-v] port

A simple extension of Python's http.server module. Serves files in the local directory, but also prints request data to the host.

positional arguments:
  port           The port to listen on.

options:
  -h, --help     show this help message and exit
  -v, --verbose  Output extra information. Use this to see all incoming request headers.
